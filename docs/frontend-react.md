## Frontend (React)

### Location

- React app: `frontend-react/`
- Entry: `frontend-react/src/main.jsx`

### Run locally

```bash
cd frontend-react
npm install
```

Create `frontend-react/.env`:

```bash
REACT_APP_API_URL=http://localhost:8000
```

Start:

```bash
npm run dev
```

Default URL: `http://localhost:8501`

### Key pages

- **Asset Discovery**: domains/subdomains/IPs/URLs management
- **Scan**: run scans (DD, API Testing, Subdomain) + Scan History + Results
- **Vulnerability**: unified findings feed (API + Subdomain)
- **Domain Insight**: domain/subdomain graph visualization
