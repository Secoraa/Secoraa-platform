# Images Directory

Secoraa logo image is stored here.

**Current logo file:** `secoraa-logo.jpg`

To replace with a new logo:
1. Place your new logo file in this directory
2. Name it `secoraa-logo.jpg` (or update the src path in Sidebar.jsx and Header.jsx)
3. Supported formats: PNG, SVG, JPG

**Recommended logo specifications:**
- Circular logo with blue/orange halves
- Circuit board pattern (left) and fingerprint pattern (right)
- Eye icon in the center
- Transparent background preferred
- Size: 32x32px minimum (will be scaled)
